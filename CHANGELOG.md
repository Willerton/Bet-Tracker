## bet-tracker V2

This release introduces major improvements to bankroll tracking, analytics, and overall usability.

### Key Updates

* Live bankroll tracking - stakes are now deducted immediately from balance
* New analytics dashboard with bookmaker breakdown and ROI tracking
* Added Active Stake and Potential Profit metrics
* Improved bet management with Money Back support and manual return overrides
* Search, filtering, sorting, and inline editing on All Bets
* Dark / Light mode toggle (saved locally)
* Cleaner layout with improved spacing and table usability

### Improvements

* Point system enhancements (cash input and rounding)
* Per-site balance tracking improvements
* Compact analytics tables (no horizontal scrolling)

### Fixes

* Resolved UI layout issues and overflow bugs
* Fixed duplicate Money Back tags
* Improved button styling in light mode
* Fixed minimise toggle behaviour
